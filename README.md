# QuickCount

A lightweight, responsive counter application with increment and decrement functionality

## Project Vision

A simple counter app with increment and decrement buttons

## Tech Stack

**Frontend**: Vanilla JavaScript + HTML + CSS
**Backend**: None (Client-side only)
**Deployment**: railway

## Features

1. **Counter Display** - Large, centered numeric display showing the current counter value with smooth transitions
2. **Increment Button** - Button to increase counter value by 1 with visual feedback on click
3. **Decrement Button** - Button to decrease counter value by 1 with visual feedback on click
4. **Reset Button** - Button to reset counter back to 0
5. **LocalStorage Persistence** - Automatically save counter value to localStorage and restore on page reload
6. **Keyboard Shortcuts** - Support arrow up/down keys for increment/decrement, and 'R' key for reset
7. **Custom Increment Value** - Allow users to set custom increment/decrement step size (default: 1)
8. **Counter History** - Track and display the last 10 counter values with timestamps
